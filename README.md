## 大纲

本次课程分为两大部分五个章节，分别为容器与微服务部分，和容器编排部分。

第一部分着重于 Docker 的介绍，共两次课

第二部分着重于 K8s 的介绍，共三次课。

通过本次课程，你将相对完整的了解 K8s 的基本知识及常用指令，并具备一定的实战能力。
